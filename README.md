# geotech

A Streamlit app for geotechnical earth pressure calculations according to DIN 4085 / Coulomb.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/paulholzapfelconsulting-art/geotech.git
   cd geotech
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Running the App

Run the Streamlit app:
```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`.

## Description

This app calculates the coefficients \( K_{agh} \), \( K_{ach} \), and \( K_{ah,min} \) for earth pressure calculations in geotechnical engineering.
